FONTLOG for the Lisu Bosa Test D fonts

This file provides detailed information on the Lisu Bosa Test D Font Software. This information should be distributed along with the Lisu Bosa Test D fonts and any derivative works.

Note that this font is a work in progress and is only suitable for testing. We do not recommend that derivative works be created at this time.

Basic Font Information

Lisu Bosa Test D is a Unicode typeface family that supports all languages that use the Lisu script and its variants.
